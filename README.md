Home NAS Lab Project

This repository documents the design, setup and operation of a personal NAS environment,
created for learning system administration and infrastructure fundamentals.

## Project Goals
- Centralized file storage
- Backup and data protection
- User and permission management
- Hands-on experience with system administration concepts

## Environment Overview
- NAS OS: OpenMediaVault 
- Storage: RAID-based configuration
- Network: Home LAN environment
- Access: Local network users

## Key Topics Covered
- Storage planning and RAID concepts
- Backup strategy and scheduling
- User access and permission management
- Basic monitoring and troubleshooting
- Documentation and operational mindset
